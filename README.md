# AR Anatomy & Engineering Visualization Learning Tool

## 📌 Overview

This project is an Augmented Reality (AR) based educational application developed using Unity. It helps students visualize complex human anatomy and engineering components in a real-world environment.

The application allows users to interact with 3D models such as organs and mechanical parts using gestures like rotate, scale, and select.

---

## 🎯 Objectives

* Provide interactive learning using AR technology
* Improve understanding of complex 3D structures
* Enable real-time visualization of anatomy and engineering models

---

## 🛠️ Technologies Used

* Unity 3D
* AR Foundation
* C#
* Android Build Support

---

## ⚙️ Features

* AR-based 3D model visualization
* Zoom, rotate, and scale models
* Human anatomy models (heart, kidney, etc.)
* Engineering components visualization
* Reset and interaction controls

## 🚀 How to Run the Project (Setup Steps)

### 🔹 Step 1: Clone the Repository

```
git clone https://github.com/your-username/your-repo-name.git
```

### 🔹 Step 2: Open in Unity

* Open **Unity Hub**
* Click **Open Project**
* Select the downloaded folder

### 🔹 Step 3: Install Required Modules

* Ensure **Android Build Support** is installed
* Enable **AR Foundation** in Package Manager

### 🔹 Step 4: Configure Build Settings

* Go to **File → Build Settings**
* Select **Android**
* Click **Switch Platform**

### 🔹 Step 5: Connect Device

* Enable **Developer Mode** on your phone
* Connect via USB

### 🔹 Step 6: Run the Application

* Click **Build and Run**
* App will install on your device

---

## 🧪 How to Use the Application (User Steps)

1. Open the app on your mobile
2. Point camera to a flat surface
3. Wait for plane detection
4. Tap on screen to place 3D model
5. Use gestures:

   * Pinch → Scale
   * Drag → Move
   * Rotate → Rotate model
6. Click buttons to switch models
7. Use reset button to clear scene
